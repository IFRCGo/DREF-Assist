## Key Improvements Made

1. **DREF Type:** Now specifies "Assessment, Response, or Anticipation" (removed old categories)
2. **Disaster Category:** References official "IFRC Crisis Categorisation"
3. **Event Description:** Explicitly requires "when, where, and what" explanation
4. **Source Information:** Must clarify if NS assessment or secondary data with sources
5. **Photos:** Must show damages/NS actions (not just any photos)
6. **NS Response:** Must clearly indicate if started and when
7. **Coordination:** Must specify mechanisms (IASC cluster system, etc.)
8. **Needs Analysis:** Must be by sector with focus on affected population needs
9. **Operation Objective:** Must define population, areas, support type, length
10. **Risk Mitigation:** Must explain mitigation for each identified risk
11. **Human Resources:** Must include Partner National Societies response units
12. **Surge Personnel:** Must specify roles and positions if requested
